dd comprehensive financing structure data to the energy_projects schema, API, Deal Tracker filters, deal detail pages, and dashboard analytics.

CURRENT STATE:
- The deal detail page shows: Deal Size, Capacity, Sector, Year Announced, Description, Deal Pipeline, Sources
- No financing type, debt/equity split, grant component, offtaker, PPA, or concessional terms are tracked or displayed
- The Drizzle ORM schema has: id, projectName, country, region, technology, status, dealSizeUsdMn, capacityMw, yearAnnounced, latitude, longitude, description, newsUrl, sourceUrl, plus dealStage, developer, financiers (investors), dfiInvolvement
- The schema may already have debtEquitySplit (text) and grantComponent (numeric) columns from Prompt 1, but they are not surfaced in any UI

REQUIREMENTS:

1. DATABASE SCHEMA — Add these columns to the energy_projects table (all nullable, use a Drizzle migration):

   financingType: text
   — Primary financing structure. One of:
   "Project Finance" | "Corporate Finance" | "Sovereign Lending" | "Concessional Loan" | "Grant / Donor Funding" | "Blended Finance" | "IPP / Concession" | "PPP / Public-Private" | "Green / Climate Bond" | "Equity Investment" | "Export Credit" | "Bilateral Aid / ODA" | "Unknown"

   financingSubTypes: text (JSON array stored as string)
   — Secondary/additional financing types when a deal uses multiple structures
   e.g., '["Concessional Loan", "Grant / Donor Funding"]' for a blended deal with DFI loan + grant top-up

   debtEquitySplit: text (if not already present)
   — e.g., "70/30", "80/20", "100/0" (for pure debt), "0/100" (for pure equity)

   grantComponent: numeric (if not already present)
   — Grant amount in USD millions, separate from the main deal size

   concessionalTerms: text
   — Description of soft terms, e.g., "IDA terms: 40yr maturity, 10yr grace, 0.75% interest"

   offtaker: text (if not already present)
   — Power purchaser or government entity, e.g., "KPLC", "ESKOM", "SENELEC", "ONEE"

   ppaTermYears: integer
   — Power Purchase Agreement duration in years, e.g., 20, 25

   ppaTariffUsdKwh: numeric
   — PPA tariff rate in USD per kWh, e.g., 0.058

   guarantor: text
   — Entity providing sovereign or partial risk guarantee, e.g., "MIGA", "Government of Kenya", "GuarantCo"

   climateFinanceTag: text
   — Climate finance classification: "Mitigation" | "Adaptation" | "Cross-cutting" | null
