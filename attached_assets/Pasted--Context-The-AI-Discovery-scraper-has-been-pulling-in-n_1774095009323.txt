## Context

The AI Discovery scraper has been pulling in new energy projects for 2023–2026. It's working well, but it's creating new sector/technology categories instead of mapping scraped projects into the 7 canonical sector buckets that the platform uses. This creates inconsistency between the Replit dev data and the Railway production data.

## The 7 Canonical Sectors

All projects in the database MUST be classified into exactly one of these 7 sectors:

1. **Solar** — solar PV, concentrated solar power (CSP), solar thermal, solar mini-grids, solar home systems, solar-battery hybrid projects
2. **Wind** — onshore wind, offshore wind, wind farms
3. **Hydro** — hydroelectric dams, run-of-river hydro, pumped-storage hydro, small hydro
4. **Natural Gas** — natural gas power plants, LNG terminals, gas pipelines, gas-to-power, gas processing
5. **Oil** — oil refineries, oil pipelines, oil-fired power, upstream oil exploration/production
6. **Geothermal** — geothermal power plants, geothermal exploration
7. **Green Hydrogen** — green hydrogen production, green ammonia, hydrogen fuel cells, electrolyzers, power-to-X projects

Modify the scraper's sector classification logic so that when it categorizes a newly scraped project, it ONLY assigns one of the 7 canonical sector values listed above. Add a mapping/normalization function like this:

```typescript
function normalizeSector(rawSector: string): string {
  const sectorMap: Record<string, string> = {
    // Direct matches
    'solar': 'Solar',
    'wind': 'Wind',
    'hydro': 'Hydro',
    'hydroelectric': 'Hydro',
    'natural gas': 'Natural Gas',
    'gas': 'Natural Gas',
    'lng': 'Natural Gas',
    'oil': 'Oil',
    'petroleum': 'Oil',
    'geothermal': 'Geothermal',
    'green hydrogen': 'Green Hydrogen',
    'hydrogen': 'Green Hydrogen',

    // Remap non-canonical categories
    'battery storage': 'Solar',        // Battery storage in Africa is predominantly solar+storage
    'battery': 'Solar',
    'storage': 'Solar',
    'ev': 'Solar',                     // EV charging infrastructure in Africa is solar-powered
    'electric vehicle': 'Solar',
    'mini-grid': 'Solar',             // Mini-grids in Africa are predominantly solar mini-grids
    'mini grid': 'Solar',
    'minigrid': 'Solar',
    'transmission': 'Natural Gas',    // Transmission/grid infrastructure — classify under the dominant fuel
    'grid': 'Natural Gas',
    'other renewables': 'Solar',      // Default: most "other renewables" in Africa are solar variants
    'biomass': 'Geothermal',          // Biomass/biogas → closest canonical fit
    'biogas': 'Natural Gas',
    'nuclear': 'Natural Gas',         // Nuclear → baseload like gas
    'coal': 'Oil',                    // Coal → fossil fuel bucket with oil
    'tidal': 'Hydro',                // Tidal/wave → water-based like hydro
    'wave': 'Hydro',
  };

  const normalized = sectorMap[rawSector.toLowerCase().trim()];
  if (normalized) return normalized;

  // Fuzzy matching fallback
  const lower = rawSector.toLowerCase();
  if (lower.includes('solar') || lower.includes('pv')) return 'Solar';
  if (lower.includes('wind')) return 'Wind';
  if (lower.includes('hydro') || lower.includes('dam')) return 'Hydro';
  if (lower.includes('gas') || lower.includes('lng')) return 'Natural Gas';
  if (lower.includes('oil') || lower.includes('petro') || lower.includes('refin')) return 'Oil';
  if (lower.includes('geotherm')) return 'Geothermal';
  if (lower.includes('hydrogen') || lower.includes('ammonia') || lower.includes('electroly') || lower.includes('h2')) return 'Green Hydrogen';
  if (lower.includes('battery') || lower.includes('storage') || lower.includes('mini-grid') || lower.includes('ev ') || lower.includes('electric vehicle')) return 'Solar';

  // If all else fails, default to Solar (most common in Africa)
  console.warn(`Unknown sector "${rawSector}" — defaulting to Solar`);
  return 'Solar';
}
```

Apply this `normalizeSector()` function wherever the scraper sets the `technology` field before inserting into the database.
